Session Setup

Paste your internal auth token and choose the user ID you want to test with. Do this first before clicking anything else.

1. Create Voice Profile

Paste: `Current User ID` from Session Setup. Type: `Display Name` and optional `Description`. Then click: `Create Voice Profile`.

2. Upload Sample And Finalize

Paste: `Voice Profile ID` from step 1 or click `Use This Voice`. Choose: any local audio file you want to test with. Then click: `Upload Sample`. After upload finishes, click: `Finalize Clone`.

Minimum Test Plan

        

          

            Test 1: Create and clone a voice
            
Save settings. Create voice profile. Upload one audio file. Finalize clone. Confirm the status becomes `ready`.

          

          

            Test 2: Generate preview speech
            
Use the same voice profile. Enter a short sentence. Generate preview. Press play on the preview audio player.

          

          

            Test 3: Queue runtime speech
            
Enter a session ID. Queue speech with user ID. Refresh runtime state. Confirm the job finishes and the runtime audio player works.

          

          

            Test 4: Cancel runtime speech
            
Queue a longer message. Immediately click cancel. Refresh runtime state and confirm the job changes to `canceled` or `interrupted`.
